## Installation Steps

1. Clone the repository

```bash
git clone https://github.com/LoopFruits/react-portfolio.git
```

2. Change the working directory

```bash
cd react-portfolio
```

3. Install dependencies

```bash
npm install
```
4. Running the application
```bash
npm start
```
